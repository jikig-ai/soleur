Provenance note (Soleur): portions of Hallmark's slop-test gate taxonomy and
anti-pattern naming were adapted into a React/Next.js Tier 1 ripgrep rule set
in plugins/soleur/skills/frontend-anti-slop/. No verbatim source code was
copied; the Soleur implementation parses a Markdown rule table, runs each
rule's regex line-by-line over `apps/web-platform/(app|components)/**/*.
{tsx,jsx,css}`, and emits findings conforming to Soleur's existing ux-audit
finding schema. The upstream attribution footer and the `build` /
`redesign` / `study` verbs are not carried over.

Original repository: https://github.com/Nutlope/hallmark
Soleur adoption tracked under: https://github.com/jikigai-ai/soleur/issues/4264
