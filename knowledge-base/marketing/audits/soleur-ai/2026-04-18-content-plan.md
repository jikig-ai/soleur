# Soleur.ai Prioritized Content Plan

## Key Findings

The three audits tell a consistent story: the technical SEO floor is effectively finished (98/A), the site's structural AEO is above the median (72/B-), and the remaining growth is **content-shaped, not infrastructure-shaped**. Three priorities dominate the next quarter:

1. **Close the core-vs-blog citation gap.** Core pages average 55/100 on AEO; the strong blog post scores 80. AI engines will continue to surface blog content over `/agents/`, `/skills/`, `/pricing/`, and `/getting-started/` until each core page carries 2+ external citations and a synthesis paragraph. This is why the AEO score is stuck in the B-range despite the homepage fix — the ceiling is now the depth tier.
2. **Own the "Claude Code plugin" query cluster before the ecosystem consolidates.** Anthropic's official marketplace now lists 101 plugins; 9,000+ exist across third-party directories. There are ~5 strong reviews of "best Claude Code plugins" ranking at the top of the SERP and Soleur is in none of them. This is the single highest-volume, highest-intent acquisition channel that matches the product's actual install path.
3. **Ship the role-based commercial-intent cluster** ("AI CTO", "AI CMO", "AI general counsel", "AI CFO"). The site has a strong pricing page comparing $95K/mo full team vs $49/mo Soleur, but no content ranks for the searches an ICP buyer actually types when evaluating a specific role replacement. This is the highest-commercial-intent traffic the site isn't capturing, and the pricing page already does the math — eight cluster pages each linking back close the loop.

- **Pillar:** `/blog/what-is-company-as-a-service/` (existing, strong — 80/B+)

### Linking rules (enforced during writing)

- Every cluster links back to its pillar in the first 200 words.
- Every cluster links to at least one sibling cluster in the body.
- Every pillar links down to every cluster in a "Related reading" section.
- All new posts link to `/pricing/` at the end (AEO audit P1: pillars currently don't).
- All new posts add a `pillar:` frontmatter field to render "Part of the X series" block (content audit P2).
