# Competitive Intelligence Report

## Executive Summary

Three material shifts since the 2026-04-01 scan. **OpenAI shipped a major Codex upgrade (April 16-17) that collapses previous platform boundaries**: Computer Use on macOS, multi-agent parallel workflows, persistent memory preview, in-app browser, image generation, scheduled long-running tasks ("across days or weeks"), and a 90+ plugin ecosystem with Atlassian Rovo, CircleCI, GitLab, and Microsoft integrations. Codex is now structurally closer to Claude Code plus Cowork combined than to the single-purpose coding agent it was six weeks ago. **Anthropic's Cowork added nine new industry-specific enterprise plugins on February 24 (equity research, investment banking, private equity, wealth management, HR, design)** bringing first-party coverage to 7+ of 8 Soleur domains. The engineering-workflow gap that was Soleur's strongest Tier 0 moat is now the only remaining Anthropic-plugin gap, and Codex just filled the equivalent gap on the OpenAI side.

In Tier 3, **a direct plugin-layer competitor emerged: `alirezarezvani/claude-skills` (235 skills, 9 domains, 12 AI-tool compatibility, explicit "Solo Founder" persona, cross-domain orchestration protocol)** — the closest structural match to Soleur seen in any scan to date, distributed through the same Claude Code plugin channel Soleur depends on. **Polsia's self-reported ARR fell from $1.5M to $450K+ between the February and April scans** — either earlier figures were inflated, or a material retention event occurred; the disclosed "most customer businesses are only weeks old" suggests revenue is churning out. **Paperclip crossed 30,000+ GitHub stars in three weeks** (double the prior scan's figure) and is now explicitly built on Claude Code. **Notion Custom Agents exited beta with 21,000+ agents built** and transitions to credit-based pricing May 4. Soleur's moat narrative ("compounding knowledge + cross-domain coherence + 8-domain breadth") is no longer unique in aspiration — two Tier 3 entrants now claim analogs. The differentiated axis that still holds is the **workflow lifecycle (brainstorm → plan → implement → review → compound)** with structured artifacts per stage, versus competitors' skill-library or task-queue models.

### Tier 0 Key Takeaways

1. **OpenAI Codex and Anthropic Cowork both collapsed to Soleur's footprint in the last ~8 weeks.** The horizontal breadth that was a structural moat is now contested at the platform level. The remaining moats are the workflow lifecycle (brainstorm → plan → implement → review → compound), the git-tracked knowledge base (vs. proprietary cloud memory), and the opinionated end-to-end choreography per domain.
2. **The engineering-workflow gap inside Cowork is the last Anthropic-plugin gap.** Soleur's highest-leverage Tier 0 positioning is still "the engineering + multi-domain CaaS that Anthropic does not ship." That gap is closeable by Anthropic at any time — plan for its closure, not against it.
3. **Founder drift to OpenAI is now a measurable risk.** business-validation.md notes founders using Cursor/VS Code/Copilot more than Claude Code. Codex's April upgrade widens that pool. A multi-platform distribution strategy (plugin for Claude Code + skills for Cursor + plugins for Codex) would address this but contradicts the current Claude-Code-plugin-first model.

### Tier 3 Key Takeaways

1. **Soleur now has a direct plugin-layer competitor (alirezarezvani/claude-skills) that claims all three Soleur moats and runs in more environments.** The "235 skills, 9 domains, 12 AI tools, Solo Founder persona, self-improving memory, cross-domain orchestration" positioning is explicitly broader than Soleur's "63 agents, 62 skills, 8 domains, Claude Code only." Soleur's remaining defensible moats are the **end-to-end workflow lifecycle with named stage artifacts** (brainstorm/plan/work/review/compound skill files with enforced handoffs), the **integrated product narrative** (Soleur as a branded platform, not a skills library), and the **git-tracked knowledge-base files** as the canonical memory layer (vs. self-improving-agent heuristics).
2. **Polsia's revenue trajectory is now ambiguous.** ARR figures moved in contradictory directions across sources. Either earlier $1.5M/2,000-company claims were inflated, or revenue is churning rapidly. Worth tracking in the next scan cycle.
3. **Paperclip + BYO-agents is the open-source path that undercuts the revenue model.** A solo founder can run `alirezarezvani/claude-skills` inside Paperclip with their own Claude API key and approximate the Soleur experience for the cost of the API. The hosted-platform revenue plan must provide meaningfully more than that combination can assemble.
4. **The plugin distribution channel is now contested.** Soleur is no longer the only CaaS-flavored Claude Code plugin, and is not the largest skills library on the marketplace.
