Theme Toggle — Sidebar Header Mock

Mock of the proposed redesign. The toggle moves from the sidebar footer to the sidebar header. Expanded: a 3-segment pill (Dark / Light / System) sits below the brand. Collapsed: the pill collapses to a single circular button that cycles modes on click. Both panels render with real --soleur-* tokens copied from globals.css so the contrast and gold accent are 1:1 with production. No app code has been touched.
