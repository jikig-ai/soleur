# feat: Create X/Twitter Banner Image for @soleur_ai

## Overview

Design and deploy a 1500x500px banner for the @soleur_ai X profile. The banner communicates Soleur's identity with the thesis statement, wordmark, and optional metrics on a brand-consistent dark background with gold accents.

## Problem Statement

The @soleur_ai X account has no banner image, leaving the profile incomplete. The banner is the primary visual real estate for visitors discovering Soleur on X.

## Proposed Solution

**Hybrid pipeline:** Generate a textured background via Gemini image generation, then composite brand-exact text via Pillow with the actual font files. This guarantees typographic fidelity that Gemini alone cannot provide.

### Pipeline

```
Pencil (.pen mockup) → Gemini (background/texture at 21:9) → Pillow (crop to 3:1 + text overlay) → Playwright (upload to X)
```

1. **Design mockup** in Pencil (.pen file) to establish layout and spacing
2. **Generate background** with Gemini (`gemini-3-pro-image-preview`, 21:9 aspect ratio, 2K resolution) — dark forge-like texture with gold accents, no text
3. **Post-process** with Pillow: center-crop from 21:9 to 3:1 (1500x500), overlay text with brand fonts (Inter, Cormorant Garamond)
4. **Upload** to @soleur_ai via Playwright MCP (headed mode, manual auth if needed)
5. **Document** banner specs in `knowledge-base/overview/brand-guide.md`

## Technical Considerations

### Aspect Ratio Mismatch

Gemini supports 21:9 (2.33:1) but not 3:1 (1500x500). Strategy: generate at 21:9 with 2K+ resolution, then center-crop vertically to 1500x500. This loses ~22% of vertical content, so the Gemini prompt must keep important elements centered vertically.

### Text Rendering

Gemini cannot guarantee Inter or Cormorant Garamond fonts. Text is rendered via Pillow's `ImageDraw` + `ImageFont.truetype()` with downloaded .ttf files from Google Fonts:
- **Wordmark:** "S O L E U R" — Inter 500, wide letter spacing, gold #C9A962
- **Thesis:** "Build a Billion-Dollar Company. Alone." — Cormorant Garamond 500, white #FFFFFF
- **Metrics (optional):** "60+ Agents · 8 Departments · 1 Founder" — Inter 400, secondary #848484

### Playwright Upload Flow

X Free tier API cannot upload banners. Use Playwright MCP in headed mode:
1. Navigate to `x.com/settings/profile`
2. Pause for manual login if session expired
3. Click banner edit area → file upload → select image
4. Save profile
5. Screenshot `x.com/soleur_ai` for verification

**Path resolution:** MCP tools resolve from repo root. Use absolute paths: `/home/jean/git-repositories/jikig-ai/soleur/.worktrees/feat-x-banner/plugins/soleur/docs/images/x-banner-1500x500.png`

### Pencil MCP Prerequisites

Per learnings:
- .pen tab must be actively visible in Cursor (WebSocket connection)
- `batch_get` before `batch_design` (never assume values)
- No auto-save — user must Ctrl+S after edits

- [ ] Uses only brand colors (#0A0A0A bg, #C9A962 gold, #FFFFFF text)
- [ ] Typography uses Inter (wordmark) and Cormorant Garamond (thesis) — not AI-approximated fonts
- [ ] Core content visible in center 60% (mobile safe zone ~900px)
- [ ] Avatar overlap area (bottom-left) has no critical content
- [ ] Banner uploaded and visible on @soleur_ai X profile
- [ ] Brand guide updated with banner specs under X/Twitter section
- [ ] Final image saved to `plugins/soleur/docs/images/x-banner-1500x500.png`

## Test Scenarios

- Given the banner image, when viewed on desktop X profile, then all text (wordmark, thesis) is fully visible
- Given the banner image, when viewed on mobile X profile (center-cropped), then the wordmark and thesis remain visible
- Given the banner image, when the X avatar overlaps bottom-left, then no text or key element is obscured
- Given the Gemini API quota is exhausted, when generating the background, then the script reports the error and exits cleanly
- Given the Playwright session is not authenticated, when navigating to X settings, then the workflow pauses for manual login

## Dependencies & Risks

| Risk | Mitigation |
|------|-----------|
| Gemini 3:1 aspect ratio unsupported | Generate at 21:9, center-crop. Ensure prompt keeps content vertically centered |
| Gemini safety filter blocks prompt | Remove commercial-sounding language from prompt. Describe visual elements, not business claims |
| X detects Playwright automation | Use headed mode (visible browser), natural interaction delays |
| Font files unavailable | Download Inter + Cormorant Garamond .ttf from Google Fonts, store in tmp/ |
| X auth session expired | Pause workflow for manual login, then resume automation |

## Implementation Phases

### Phase 1: Design and Generate

1. Verify `GEMINI_API_KEY` is set
2. Download Inter and Cormorant Garamond .ttf fonts from Google Fonts
3. Create Pencil mockup (`.pen` file) at 1500x500 layout for visual reference
4. Generate background texture via Gemini (`gemini-3-pro-image-preview`, 21:9, 2K)
   - Prompt: dark forge atmosphere, gold gradient accents at edges, no text, brand colors only
5. Center-crop from 21:9 to 1500x500 with Pillow
6. Overlay text with Pillow using brand fonts:
   - "S O L E U R" wordmark (Inter 500, gold, center-top area)
   - "Build a Billion-Dollar Company. Alone." (Cormorant Garamond 500, white, center)
   - Try metrics "60+ Agents · 8 Departments · 1 Founder" (Inter 400, secondary gray) — remove if cluttered
7. Save to `plugins/soleur/docs/images/x-banner-1500x500.png`

### Phase 2: Upload to X

1. Screenshot current @soleur_ai profile state (backup)
2. Navigate to X settings via Playwright MCP (headed mode)
3. Upload banner image, save profile
4. Screenshot updated profile for verification

### Phase 3: Document and Finalize

1. Add banner specs to brand guide under `### X/Twitter` in `## Channel Notes`
2. Run `soleur:compound` to capture learnings
3. Commit all artifacts and push
