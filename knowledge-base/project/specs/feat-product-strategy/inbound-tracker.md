# Inbound Interest Tracker

**Campaign:** Case Study Distribution (2026-03-12 to 2026-04-01)

**Last updated:** 2026-03-11
**Review cadence:** Weekly (Fridays)

## Signal Definitions

| Type | Definition | Example |
|------|-----------|---------|
| DM | Direct message on any platform requesting access or asking how to try Soleur | X DM: "This looks interesting, how do I get started?" |
| Reply | Public reply on social post indicating interest beyond casual engagement | Reddit comment: "Would love to try this for my startup's legal docs" |
| Star | GitHub star from a non-creator account (indicates awareness + bookmark intent) | github.com/jikig-ai/soleur starred by external user |
| Question | Substantive question about capabilities, architecture, or use cases | IH comment: "Does this work for non-technical founders too?" |
| Install | Someone installs the plugin or follows setup instructions | Discord: "Just installed, running my first brainstorm" |
| Join | Joins Discord server from social referral | New Discord member who mentions seeing a post |

## Signal Strength

| Strength | Definition | Follow-up Priority |
|----------|-----------|-------------------|
| Cold | Passive engagement -- like, upvote, retweet without comment | None. Track for volume metrics only. |
| Warm | Active engagement -- substantive reply, question, or share with commentary | Reply within 24h. Invite to Discord if conversation extends. |
| Hot | Explicit interest -- DM requesting access, "how do I try this," install attempt | Reply within 4h. DM with onboarding. Log for interview pipeline. |
