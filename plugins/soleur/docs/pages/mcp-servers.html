MCP Servers

External tool integrations via the Model Context Protocol.

Context7

context7 MCP

Retrieves up-to-date documentation and code examples for any programming library or framework. Provides two tools: resolve-library-id to find the correct library identifier, and query-docs to fetch relevant documentation snippets.

How it works

MCP (Model Context Protocol) servers extend Claude Code's capabilities by providing access to external tools and data sources. Soleur bundles Context7 as a built-in MCP server that is automatically available when the plugin is installed.

When you ask about a library or framework, Context7 searches its index of documentation and returns relevant code examples and API references. This ensures Claude Code always has access to current documentation, even for recently released versions.

Usage

Context7 is invoked automatically when Claude Code needs documentation lookup. You can also trigger it explicitly:

// Claude Code will use Context7 to fetch the latest docs
"How do I set up authentication in Next.js 15?"

// Or ask for specific library documentation
"Look up the Rails 8 Active Record encryption API"

Configuration

Context7 is pre-configured in the plugin manifest and requires no additional setup. The MCP server runs automatically when needed.

// From plugin.json
"mcpServers": {
  "context7": {
    "type": "stdio",
    "command": "npx",
    "args": ["-y", "@upstash/context7-mcp"]
  }
}
