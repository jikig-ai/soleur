context7

HTTP

Retrieves up-to-date documentation and code examples for any programming library or framework via Context7.

URL: https://mcp.context7.com/mcp

Tools

  • resolve-library-id -- Resolves a package name to a Context7-compatible library ID
  • query-docs -- Retrieves documentation and code examples for a given library
