Getting Started

Install Soleur and start building smarter.

Installation

claude plugin install soleur

The Workflow

Soleur follows a structured 5-step workflow for software development:

1. /soleur:brainstorm

Explore what to build

2. /soleur:plan

Create a structured implementation plan

3. /soleur:work

Execute the plan with quality checks

4. /soleur:review

Run multi-agent code review

5. /soleur:compound

Document learnings for future reference

Quick Commands

/soleur:one-shot

Full autonomous pipeline from plan to PR

/soleur:help

List all available components

/soleur:sync

Analyze and document your codebase

Common Workflows

Building a Feature

brainstorm → plan → work → review → compound

Fixing a Bug

work (directly on the fix) → review → compound

Reviewing a PR

review (run multi-agent review on existing PR)
