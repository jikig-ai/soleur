Installation

The Workflow

Soleur follows a structured 5-step workflow for software development:

  1. /soleur:brainstorm Explore what to build
  2. /soleur:plan Create a structured implementation plan
  3. /soleur:work Execute the plan with quality checks
  4. /soleur:review Run multi-agent code review
  5. /soleur:compound Document learnings for future reference

Quick Commands
