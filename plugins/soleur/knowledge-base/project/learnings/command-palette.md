# Using the command palette (Ctrl+K / ⌘K)

The command palette is the fastest way to do anything in Soleur — think of it as a search box for actions and destinations.

## Opening it

- **Windows/Linux:** press **Ctrl+K**
- **Mac:** press **⌘K**

## What you can do from the palette

- **Go anywhere** — type the name of a section (Dashboard, Inbox, Workstream, CRM, Knowledge Base, Routines, Releases) and press Enter.
- **Ask an agent** — start a conversation with your AI agent directly.
- **Open Settings** — the Settings entry drills into All settings, Team, Billing, and Audit log.
- **Toggle the sidebar** — collapse or expand the navigation rail.
- **See keyboard shortcuts** — opens the shortcut reference.

## How it works

1. Open the palette.
2. Start typing — results filter as you type.
3. Use the arrow keys to highlight a result, then press **Enter**.
4. Press **Escape** to close without doing anything.

Each destination in the palette shows its keyboard shortcut next to it, so you gradually learn the fast paths (for example, Dashboard shows `G D`).
