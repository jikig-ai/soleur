# Keyboard shortcuts reference

Press **?** (question mark) anywhere in the app to open the live shortcut overlay. This page summarizes the same shortcuts.

## The essentials

| Shortcut | Action |
|---|---|
| **Ctrl+K** / **⌘K** | Open the command palette |
| **?** | Show the keyboard shortcuts overlay |
| **Escape** | Close the palette or overlay |

## Go-to sequences (press `g`, then the letter)

These are two-key sequences: tap **g**, release, then tap the second key.

| Sequence | Destination |
|---|---|
| `g` `d` | Dashboard |
| `g` `i` | Inbox |
| `g` `w` | Workstream |
| `g` `m` | CRM |
| `g` `k` | Knowledge Base |
| `g` `r` | Routines |
| `g` `l` | Releases |
| `g` `c` | Ask an agent |

## Direct accelerators (hold Ctrl / ⌘)

Some destinations also have a one-chord accelerator:

| Shortcut | Destination |
|---|---|
| **Ctrl+D** / **⌘D** | Dashboard |
| **Ctrl+I** / **⌘I** | Inbox |
| **Ctrl+R** / **⌘R** | Routines |

The app shows **⌘** on Apple devices and **Ctrl** elsewhere — the shortcuts are the same.

## Notes

- Shortcuts are paused while a dialog is open, so typing in a form never accidentally navigates.
- If a `g` sequence doesn't fire, make sure focus isn't inside a text field.
