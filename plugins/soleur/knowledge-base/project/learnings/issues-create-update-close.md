# Creating, updating, and closing work items

Work items (issues) live on the **Workstream** board (`g` then `w`). You can manage them directly from the board.

## Creating an item

1. Go to Workstream.
2. Use the create action on the board.
3. Give the item a clear title and any details, then save. It appears in the appropriate status column.

## Updating an item

- Open the item from its card to edit the title, description, or status.
- Changing the status moves the card to the matching column.

## Closing an item

- Closing is how you remove an item from active work — items are closed rather than deleted, so history is preserved.
- Close an item when it is finished or no longer relevant.

## Good to know

- Changes you make on the board are recorded, so there's always an audit trail of what changed.
- Your AI agent can do all of this for you too — ask it in a conversation ("create an issue for…", "close the item about…"). Anything you can do on the board, the agent can do as well.
